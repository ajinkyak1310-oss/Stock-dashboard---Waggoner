# Portfolio Stock Dashboard

A portfolio tracking dashboard built with Streamlit and yfinance.

## Features
- Live stock prices and % change
- Click any stock for detailed analysis
- DCF Valuation
- Comparable Valuation using industry peers
- Portfolio Heatmap
- Financial statements (P&L, Balance Sheet, Cash Flow)
